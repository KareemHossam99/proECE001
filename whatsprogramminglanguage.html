//What's programming language:

it's a group of commands, instructions and other terms use to make software programs to do a certain task or make it easy to do tasks in a short time. Programs are made through languages called "high-level programming languages" which is compiled into "low-level programming languages" using the compiler.

-a compiler is a computer program that translates high-level languages into a machine language
-high-level languages are designed to be easy for the programmers to deal with, read, understand and write using logical words and symbols which is reserved in most programming languages like function, while, if, and else and symbols like <, >, ==, and != most popular high level languages are c++ and java
-low-level languages are the machine languages and the assembly of the program which include a list of commands
and instruction which are more difficult and complicated than
the higher-level programming languages

Examples on programming languages:

  • C++
  • C#
  • python
  • ruby
